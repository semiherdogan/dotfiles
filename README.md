# dotfiles
My personal dotfiles.

Usage

- Source `bootstrap.sh` file into `~/.zshrc` or `~/.bashrc` file. See below:
```
source ~/dotfiles/bootstrap.sh
```

- Add this line into your `~/.gitconfig` file.
```
[include]
	path = "~/dotfiles/gitconfig"
```
